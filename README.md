# Royal Spicy Masala

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```
bash
# Clone the repository
git clone https://github.com/yourusername/RoyalSpicyMasala.git

# Navigate to the project directory
cd RoyalSpicyMasala

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```
bash
npm run build
```
